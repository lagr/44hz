WebFEBoilerPlate
================

Note:
I created this project some time ago. Today, you should probably just use http://yeoman.io/

Simple front-end boiler plate. Has HAML, SASS, Coffee and Bower integration via Gulp.

Pre-requisites:
  - node (and npm)
  - coffee script
  - sass
  - haml
  - gulp
  - bower

To Setup:
  - $ npm install
  - $ bower install

To Build:
  - $ gulp
